01Keyword filters

Whole-word, case-insensitive. A job is dropped if its title hits any exclude keyword, and kept only if its title or company hits at least one include keyword.

02USAJobs

Official US federal jobs API. Needs the USAJOBS_API_KEY and USAJOBS_EMAIL repo secrets — free key from developer.usajobs.gov. Skipped gracefully if unset.

03Greenhouse

Public board API, no key. The board token is the last part of a company’s boards.greenhouse.io/<token> or job-boards.greenhouse.io/<token> careers URL.

04Lever

Public postings API, no key. The company slug is the last part of a jobs.lever.co/<company> careers URL (e.g. palantir).

05RSS / Atom feeds

Any jobs RSS or Atom feed. The label is shown as the company/origin on the board.

06Save & deploy

GitHub Pages is static, so this page can’t write to the repository. Saving is a commit:

  1. Press Copy config.json below.
  2. Press Edit on GitHub, select everything in the editor, paste, and commit to main.
  3. The Aggregate jobs workflow re-runs automatically on that commit and the board refreshes.
